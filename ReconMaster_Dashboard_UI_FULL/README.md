# ReconMaster (UI + Desktop)

## What is included
- FastAPI UI (Controller/)
- Admin authentication (Admin/admin.conf)
- Legal gate (Legal/)
- Desktop wrapper (Desktop/desktop_app.py) -> runs UI in a native window (no browser)

## First run (Kali)
1. Create venv
2. Install requirements
3. Set admin password hash
4. Start desktop app

See `Build/README_BUILD.md`.
